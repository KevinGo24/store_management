# PROYECTO INTEGRADOR

buenas equipo, este es el proyecto integrador en lo cual vamos hacer de productividad un sistema de inventarios que estara relacioanda al comercio y al tema de los inventarios, para que el consumidor tenga su propio inventario desde una pagina web o aplicativo para que no lo haga de forma manual si no ya un poco mas moderna 

## HERAMIENTAS DE DESARROLLO

- HTML
- CSS
- VITE
- POSTGRES SQL
- bootstrap-icons

## EJECUTAR PROYECTO

- npm rum dev
- npm init -y ( debe ser obligatorio para poder correr el proyecto )
